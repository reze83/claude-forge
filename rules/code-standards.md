# Code-Standards
- Moderne Syntax: ES2024+, Python 3.12+, aktuelle Rust Edition
- Tests fuer jedes neue Feature (TDD bevorzugt)
- Funktionen max. 50 Zeilen — bei mehr aufteilen
- Typisierung: TypeScript strict, Python Type Hints
- Keine Magic Numbers — Konstanten definieren
- Error Handling: Keine stillen Fehler, immer explizit
