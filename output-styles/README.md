# Output Styles

This directory contains custom output style definitions for Claude Code.

Output styles allow plugins to define how Claude Code formats its responses,
including custom rendering templates and formatting rules.

## Usage

Place `.md` or `.json` style definition files in this directory.
They will be registered via `plugin.json` under the `outputStyles` key.

## Documentation

See the [Claude Code Plugins Reference](https://code.claude.com/docs/en/plugins) for details on output style format and capabilities.
