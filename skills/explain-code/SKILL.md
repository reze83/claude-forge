## Ablauf

1. **Analogie**: Alltags-Vergleich der das Konzept greifbar macht
2. **ASCII-Diagramm**: Visueller Ablauf / Datenfluss
3. **Schritt-fuer-Schritt**: Zeile fuer Zeile mit Erklaerung
4. **Stolperfallen**: Haeufige Fehler und Missverstaendnisse
